Creating Splits

Key Action Result
prefix + | Split horizontally Panes side-by-side
prefix + - Split vertically Panes stacked
prefix + _ Split vertically Same as - (oh-my-tmux)

Horizontal Split (|)

┌─────────┬─────────┐
│         │         │
│ Pane 1  │ Pane 2  │
│         │         │
└─────────┴─────────┘

Vertical Split (-)

┌───────────────────┐
│      Pane 1       │
├───────────────────┤
│      Pane 2       │
└───────────────────┘
Note: Splits preserve current directory
Key concepts: - Horizontal split (|) creates side-by-side panes - Vertical split (- or _) creates stacked panes - Split naming is based on the divider line, not the layout - New panes inherit the current working directory - oh-my-tmux provides both - and _ for vertical splits Demo: Create a horizontal split, then split one pane vertically to show complex layouts

Pane Operations

KeyAction
prefix + xKill pane (no confirm - custom)
prefix + XKill window (no confirm - custom)
prefix + zToggle pane zoom (fullscreen)
prefix + !Break pane into new window
prefix + +Maximize pane to new window (oh-my-tmux)
prefix + SpaceCycle layouts
prefix + \Rotate/cycle layouts
prefix + gJoin pane from window (vertical, fzf)
prefix + GJoin pane from window (horizontal, fzf)
Pane operations for managing individual panes: - Kill pane (x) and kill window (X) are customized to skip confirmation prompts - Zoom (z) toggles fullscreen for the current pane - press again to unzoom - Break pane (!) moves a pane to its own new window - Maximize (+) is an oh-my-tmux feature similar to zoom but creates a temporary window - Cycle layouts (Space) rotates through different pane arrangement presets

Joining Panes

Join pane merges a pane from another window into the current window

KeyAction
prefix + gJoin pane vertically (stacked)
prefix + GJoin pane horizontally (side-by-side)
Tip: Uses fzf popup to select source window. Opposite of prefix + ! (break pane)
Joining panes allows you to consolidate panes from different windows into one view. - lowercase g joins vertically, creating stacked panes - uppercase G joins horizontally, creating side-by-side panes - An fzf popup appears to interactively select which window's pane to bring in - This is the inverse operation of break pane (prefix + !) which moves a pane out to its own window - Useful for reorganizing your workspace without copying commands or losing terminal state

Swapping Panes

KeyAction
prefix + > Swap current pane with next (oh-my-tmux)
prefix + < Swap current pane with previous (oh-my-tmux)
Swapping panes allows you to reorder panes without recreating your layout. The greater-than and less-than symbols indicate the direction of the swap. These are oh-my-tmux bindings - standard tmux uses curly braces for swapping.

Resizing Panes

KeyAction
prefix + HResize left (repeatable, oh-my-tmux)
prefix + JResize down (repeatable)
prefix + KResize up (repeatable)
prefix + LResize right (repeatable)
Tip: Press prefix once, then H/J/K/L multiple times (repeat mode)
These are custom oh-my-tmux bindings that use uppercase letters (Shift + h/j/k/l) for resizing panes. The repeat mode is very convenient - you only need to press the prefix once, then you can resize continuously by pressing H, J, K, or L multiple times without repeating the prefix each time. This makes fine-tuning pane sizes much faster.

Layout Presets

KeyAction
prefix + SpaceCycle through layouts
prefix + Alt-1Even horizontal
prefix + Alt-2Even vertical
prefix + Alt-3Main horizontal
prefix + Alt-4Main vertical
prefix + Alt-5Tiled
Tmux provides several preset layouts that automatically arrange panes. The Space key cycles through all available layouts, which is great for experimentation. Alt-1 through Alt-5 jump directly to specific layouts: even horizontal and vertical split panes equally in their respective directions, main layouts dedicate most space to one primary pane with others arranged horizontally or vertically, and tiled creates a grid. These presets save time when you need standard arrangements quickly.

Exercise 4: Panes

  1. Split vertically: prefix + |
  2. Split horizontally: prefix + -
  3. Navigate with prefix + h/j/k/l
  4. Resize a pane: prefix + H (repeat)
  5. Zoom a pane: prefix + z (toggle back)
  6. Swap panes: prefix + >
  7. Kill a pane: prefix + x
Practice creating and managing panes within a window. Start by splitting vertically, then horizontally to create multiple panes. Use vim-style navigation to move between them. Try resizing with H/J/K/L. Zoom in on a single pane with z, then toggle back. Experiment with swapping pane positions and closing individual panes.
