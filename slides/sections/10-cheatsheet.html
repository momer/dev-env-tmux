Cheat Sheet

Quick reference for all essential keybindings

This section provides a comprehensive cheat sheet organized by category. Use these slides as a quick reference during or after the tutorial. Each category is color-coded for easier navigation.

Quick Reference (1/2)

Prefix: Ctrl-a or Ctrl-b

Sessions

KeyAction
prefix + sSession tree
prefix + dDetach
prefix + $Rename
prefix + QSave & quit

Windows

KeyAction
prefix + cNew window
prefix + n/pNext/prev
prefix + 0-9Jump to number
prefix + XKill window

Panes

KeyAction
prefix + |Vertical split
prefix + -Horizontal split
prefix + h/j/k/lNavigate
prefix + xKill pane
prefix + zZoom toggle
prefix + \Rotate layout
prefix + g/GJoin pane
This slide covers the three main organizational levels of tmux: sessions, windows, and panes. Each category is color-coded for easier reference. Sessions (blue): Managing your workspaces - switching, detaching, renaming, and safely quitting with save. Windows (green): Tab-like navigation - creating new windows, jumping between them, and removing them. Panes (purple): Split screen management - creating splits, navigating between them, zooming for focus, and advanced operations like rotating layouts or joining panes from other windows. Note the join pane bindings (g/G) use fzf for selection, making it easy to merge panes from any window.

Quick Reference (2/2)

Copy Mode (prefix + [)

KeyAction
j/kLine up/down
C-j/C-kPage up/down
/Search
vStart selection
yCopy
prefix + ]Paste

Persistence

KeyAction
prefix + Ctrl-sSave
prefix + Ctrl-rRestore

Config & Plugins

KeyAction
prefix + eEdit config
prefix + rReload config
prefix + mToggle mouse
prefix + IInstall plugins
prefix + ?List all bindings

Discovery

KeyAction
prefix + Spacewhich-key menu
prefix + ?List all bindings
Pro Tip: Press prefix + Space to open the which-key menu for interactive command discovery, or prefix + ? for a complete binding list.
This slide covers the remaining essential categories: Copy Mode (orange): Vim-style navigation in scrollback buffer. Enter with prefix + [, navigate with j/k, search with /, select with v, copy with y, and paste with prefix + ]. Persistence (light blue): Manual save and restore commands for session state. These work alongside continuum's automatic saving. Config & Plugins (red): Quick access to editing and reloading configuration, toggling mouse mode, and managing plugins with tpm. Discovery (green): The two most important discovery tools - which-key for interactive menus and the native tmux binding list. These help you learn and remember keybindings without leaving tmux. The tip emphasizes which-key as the primary discovery tool, with the native list as a fallback for comprehensive reference.
